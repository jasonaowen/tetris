# Tetris

An implementation of the classic game using C++ and SDL.

## Building

This project uses CMake.

```sh
$ mkdir build
$ cd build
$ cmake -G "Unix Makefiles" ..
$ make catch
$ make && ./Test && ./Tetris
```
